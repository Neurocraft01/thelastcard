## Troubleshooting

### Can't Login
- Verify email is correct
- Check for typos in password
- Use "Forgot Password" to reset
- Check spam folder for reset email
- Contact support if locked out

### Profile Not Updating
- Hard refresh browser (Ctrl+F5)
- Clear browser cache
- Try incognito/private mode
- Check if changes were saved

### Order Issues
- Check "My Orders" for latest status
- Tracking number typically available in 2-3 days
- Production time: 3-5 business days
- Shipping time: 5-7 business days
- Contact support for urgent issues

### QR Code Not Working
- Ensure phone camera has internet
- Try different QR code scanner app
- Verify profile URL works in browser
- Download new QR code from dashboard

### Upload Fails
- Check file size (max 5MB for images)
- Use supported formats (JPG, PNG, SVG)
- Ensure stable internet connection
- Try resizing image if too large
